# Tsuki (⚠️ WIP ⚠️)

Tsuki is a simple functional programming language that targets WebAssembly bytecode.

# Features

A rough sketch of language features

- [ ] Literals
- [ ] Variables
- [ ] `if`/`else` statements
- [ ] Functions

- [ ] Function capturing
